HALLOWEEN PROJECT

Hey~, it's been a little while since I've worked in any project, after, well, cancelling MicTerror, you know it was a good project idea, but unfortunately I wasn't having any motivation to work on it, but hey, I tried lmao. Anyways, to the main point, the project, WELLLLLL I'll be repeating the same thing I've done in the past 2 years, work in another copy of Super Platform, but that's my replacement for MicTerror, it was meant to be a hallowen special project rather than a simple SM4LL, since I cancelled it I replaced it with this Super Platform remake, since it is already obvious what's the project about, you'll be seeing the name either here or on the upcoming teaser that I'll work on soon, the project's name is... Super Platform: Reoriginal!, pretty original right? (irony), anyways, it's been a while I've wanted to work on a remake of one of my most favorite projects and the first one to exist after I left Roblox for good. So here it is, hope you guys like it when it gets released on October 31th. This project will take all the ideas shown in the original Super Platform and make a "scarier remake" of it, but it won't be canon to the actual game's story, which started with a demo last year.

Anyways, have fun when it gets released, it is really simple but I'm quite loving working on it, cya lads and see you on the teaser's release (which should come out on October 30th).
Oh, and also the teaser and the build itself should be scheduled to these dates since I have some stuff to do with my sister on these dates, so everything should be done by the end of tomorrow.
